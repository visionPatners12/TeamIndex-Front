Overall goal:
Create a landing page for a “Mainnet Pilot” launch of Team Index.

Product concept:
Team Index is a new way to back a team beyond a single match.
Instead of placing a one-off bet, users enter a team-based pool represented by a live token.
The underlying funds of each Team Index are deployed on Polymarket through the index strategy.
Users can enter with:
- USDC on Polygon and receive the core token on Polygon
- CHZ or supported fan tokens and receive the wrapped Team Index token
In case of victory, gains are distributed to eligible holders who hold the required amount of the token.
The token is not publicly listed yet, so display “Listing Soon”.
During the pilot, token value is derived from the live index value.
